Tree Data Input (For Accurate Carbon Calculation)

Choose one method: Upload CSV/Excel file with tree data OR manually enter average measurements. Carbon credits will be calculated as: Number of Trees × Average Tree Carbon Sequestration

Option 1: Upload Tree Data File

Upload a CSV or Excel file containing individual tree measurements. Required columns: height, dbh (diameter). Optional: age, species

Expected file format (click to expand)
CSV/Excel columns (case insensitive):
• height - Tree height in meters or centimeters
• dbh or diameter - Diameter at breast height in meters or centimeters
• age (optional) - Tree age in years
• species (optional) - Tree species name

Example:
height,dbh,age,species
8.5,0.35,15,Mangrove
9.2,0.42,18,Rhizophora
7.1,0.28,12,Avicennia
